## 这是什么

因为低版本的Minecraft无法输入中文, 这是一个Lunar Client的修改版, 可以解决中文修复问题

## 快捷方式教程

创建快捷方式不要把主程序`Lunar Client.exe`做成快捷方式, 这样做会使程序出现bug

----------------↓正确方法↓---------------------

1. 将`launchlc.bat`弄成快捷方式
2. 右键->属性->自定义->更改图标->选择LC的主程序-> 选择第一个图标->点击确定
